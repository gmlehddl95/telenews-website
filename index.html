주요 기능

실시간 뉴스 알림

관심 키워드를 등록하면 10분마다 자동으로 새로운 뉴스를 체크하여 알려드립니다.

  • 키워드 기반 맞춤 뉴스
  • 논리 연산 지원 (AND/OR)
  • 중복 뉴스 자동 필터링
  • 네이버 뉴스 API 연동

주가 모니터링

나스닥 100(^NDX)과 TQQQ의 실시간 가격을 추적하고 급락 시 즉시 알려드립니다.

  • 나스닥 100 지수 전고점 대비 하락률 알림
  • 5%부터 1%p 단위로 하락 시 최초 1회 알림 (예: 5%, 6%, 7%...)
  • 정각부터 2시간마다 자동 체크
  • TQQQ 추가 하락 시나리오 제공

방해금지 시간

원하는 시간대에는 알림을 받지 않도록 설정할 수 있습니다.

  • 버튼으로 간편하게 설정
  • 추천 시간대 제공
  • 수동 명령어는 언제나 사용 가능
  • 자동 알림만 일시 중지

사용 방법

1

봇 시작하기

텔레그램에서 @MOEFNewsBot을 검색하고 /start 명령어를 입력하세요.

2

키워드 등록

➕ 키워드 추가 버튼을 눌러 관심 있는 뉴스 키워드를 등록하세요.
💡 논리 연산도 가능합니다: "삼성 AND 전자", "(속보 OR 긴급) AND 코스피"

3

알림 받기

이제 자동으로 뉴스와 주가 정보를 받아보세요!
📰 뉴스 확인으로 즉시 확인도 가능합니다.

명령어

📰 뉴스 관련

/add 키워드 추가
/list 키워드 목록
/news 즉시 뉴스 확인

📊 주가 관련

/stock 나스닥 100/TQQQ 정보

🔕 방해금지

/setquiet 시간대 선택
